# Postman Collection execution

## Steps to Run

1. Import DummyJSON_Collection.json in Postman.
2. Import DummyJSON_Environment.json in Postman.
3. Select DummyJSON_Environment as environment.
4. Click Run on collection.
5. Select run collection as manually and iteration as 1.
6. Run and Execute all requests.

## Base URL
https://dummyjson.com

## Notes
- This collection validates Auth, Products, Cart, and Orders APIs.
- Includes positive, negative, and edge cases.
