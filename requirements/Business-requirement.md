## Business requirements:

**BR-01:** PRODUCT BROWSING


1.The system should allow users to view a list of available products.


2.Users should be able to view details of a selected product.


------------
**BR-02:** PRODUCT SEARCH AND CATEGORIES


1.The system should allow users to search for products.


2.The system should allow users to view products by category.


------------
**BR-03:** USER LOGIN


1.The system should allow registered users to log in.


2.The system should prevent access to certain features without login.


-------------
**BR-04:** SHOPPING CART
  
  
1.The system should allow users to add products to a shopping cart.


2.Users should be able to view and manage items in the cart.


-------------
**BR-05:** CHECKOUT
  
  
1.The system should allow users to proceed to checkout with items in the cart.


2.The system should show order confirmation after checkout.


--------------
**BR-06:** ERROR HANDELING


1.The system should display appropriate messages when an action fails.


2.The system should handle invalid inputs gracefully.
